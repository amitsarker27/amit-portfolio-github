# Amit Sarker — Portfolio (GitHub Pages Ready)

Pure HTML / CSS / JavaScript — no build step. Deploy directly to GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `amit-profile.jpg`

## Deploy to GitHub Pages (5 minutes)

1. Create a new GitHub repo, e.g. `amit-portfolio` (or `<your-username>.github.io` for a root URL).
2. Upload all 4 files to the repo root (drag & drop on github.com → "Add file" → "Upload files" → Commit).
3. Go to repo **Settings → Pages**.
4. Under **Source**, choose **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → **Save**.
5. Wait ~1 minute. Your site is live at:
   - `https://<your-username>.github.io/amit-portfolio/`
   - or `https://<your-username>.github.io/` if you used the `<username>.github.io` repo name.

## Local preview
Just open `index.html` in your browser. Done.
